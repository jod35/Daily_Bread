# Daily Bread


## API 

| ROUTE | METHOD | DESCRIPTION |
| ------| ------- |------------|
|/verses | GET     | get all records |
|/verses | POST     | create a verse resource |
| /verse/id | GET    | get a verse with an id |
| /verse/id  | PUT    | update a verse with an id |
| /verse/id   | DELETE | delete a verse with an id |
